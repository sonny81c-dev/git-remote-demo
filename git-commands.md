# Git commands learned so far

- `git init` – initialize a new local repository
- `git status` – show the current repository state
- `git add <file>` – stage a file for the next commit
- `git add -A` – stage all changes
- `git commit -m "message"` – create a commit with a message
- `git diff` – show unstaged changes
- `git log --oneline` – show commit history
- `git remote -v` – show connected remote repositories
- `git push` – upload local commits to GitHub
- `git pull` – download and integrate remote changes
- `git fetch` – download remote changes without merging
- `git clone <repo-url>` – create a local copy of a remote repository
- `git commit --amend` – modify the most recent commit
- `git checkout -- .` – discard local unstaged changes in tracked files
